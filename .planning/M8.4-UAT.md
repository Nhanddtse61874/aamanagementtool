## 🔴 READ THIS FIRST — you get ONE chance at the password

**Your admin row has no password.** Migration v10 promoted the first user (`Admin`) to admin, but gave them no `password_hash` — so today nobody can log in and nobody can set one.

On the API's **first start**, it generates a random password, applies it, and prints it to the console **exactly once**:

```
ADMIN PASSWORD GENERATED (shown ONCE — copy it now)
  username: Admin
  password: ....
```

**Copy that line.** There is no second chance and no way to recover it — deliberately: it exists only in that one log line, so there is no secret sitting in a file to leak. If you miss it, the fix is to null the `password_hash` column and restart.

## Things that changed in the UI, that you did not ask for

**Three buttons were removed** from Log Work because they were **fake**:
- **"+ Add task"** — called `toast.show('Task added')` and added nothing.
- **"Move to next month"** — no handler at all.
- **"Drop a task here to delete"** — no drop handler.

None has an endpoint in M8.4. A button that *says* "Task added" while adding nothing is worse than no button. **Say the word and they come back in one commit.**

**Smart Fill is a minimal panel** — one task, pick days, total hours, split evenly. WPF's version also has backlog-code search, multi-task checkboxes, a `FillFull8h` mode and a preview. This is the smallest thing that is *real* rather than a fake button.

## Still open — and it is yours to decide, not a wave's

**How does this reach a colleague's browser?**

The dev loop works on your machine. Production does not exist yet, and it collides with the blocker you deferred: *"the company has no server."* The two shapes are:

- **One designated workstation** runs the API **and serves the Angular files from the same origin** (one port, one process). Everyone else points a browser at it. This is what the spec assumes, and it is what makes SQLite's single-writer property hold.
- **Everyone runs their own local copy** against the shared-drive database. This is what the WPF app does today, and it is precisely the N-writers-over-SMB shape sqlite.org warns against.

I am not going to pick for you. But **nobody else can use this until it is picked.**
