## Adjacent losses noticed while tracing (outside the claim list, flag for whoever owns them)

- **Gantt empty state.** `TaskListTab.xaml.cs:377-388` draws "No dated backlogs to chart for this month." when the model is null or the axis is empty. The web renders an empty card and says nothing (`.html:78-80` — `@if (gantt(); as g)` with no `@else`).
- **Gantt axis weekday labels.** WPF prints day-of-month *and* the weekday abbreviation under it (`xaml.cs:257-277`); the web prints day-of-month only, with the full ISO date as a `title` (`component.ts:720-722`, `.html:84`).
- **`BackfillMissingMonthsAsync`** (`App.xaml.cs:104`) is a startup lifecycle behaviour with no web equivalent — see the Export row.
