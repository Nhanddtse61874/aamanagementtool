## Summary

Every USR-0x behavior described in A7 has a 1:1 (or superset) route + Angular UI equivalent. The two "no-Edit / no hard-delete" WPF constraints are absences, not features to preserve, and both hold true (or are exceeded, in Edit's case, without regressing anything) in the web app as well. Deleting `src/TimesheetApp/` cannot lose any Users-tab behavior.
